html

(Voltar)

Tag padrão do Html, como o nome já diz define é raiz do documento.

header

(Voltar)

Tag padrão do Html, contem padroes/metodos e infromações para o documento.

title

(Voltar)

Tag padrão do Html, define o title do documento.

body

(Voltar)

Tag padrão do html, ela é usado para marcar o corpo do documento.

h1,...,h6

(Voltar)

Essas tags delimitam os niveis de importancia do assunto principal de um contedo sendo o "h1" mais importe e "h6" menos importante.

hr

(Voltar)

Essa tag geralmente é usada para quebrar linhas no html.

p

(Voltar)

Essa tag define e delimita um paragrafo no documento.

ul

(Voltar)

Essa tag define uma lista não ordenada

li

(Voltar)

Define os itens em uma determinada lista, geralmente contidas pelas tags "ul" ou "ol".

ol

(Voltar)

Essa tag define uma lista ordenada.

a

(Voltar)

Essa tag define um link ou hyperlink juntamente com "href".

small

(Voltar)

Essa tag serve definir um texto menor.
